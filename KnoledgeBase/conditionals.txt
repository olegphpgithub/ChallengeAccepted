Приведи пример смешанного условного предложения. Предложение должно быть в инверсной форме.
Сочетай 2-й и 3-й тип.


| Тип условного предложения             | Условие (If-clause)     | Результат (Main clause)          | Формула                                          | Пример                                                                                                                       |
| ------------------------------------- | ----------------------- | -------------------------------- | ------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| **Нулевой тип (Zero Conditional)**    | **If + Present Simple** | **Present Simple**               | If + Present Simple → Present Simple             | **If** you heat water to 100°C, it **boils**. (Если нагреть воду до 100°C, она закипит.)                                     |
| **Первый тип (First Conditional)**    | **If + Present Simple** | **will + Infinitive**            | If + Present Simple → will + Infinitive          | **If** it rains tomorrow, we **will stay** home. (Если завтра пойдет дождь, мы останемся дома.)                              |
| **Второй тип (Second Conditional)**   | **If + Past Simple**    | **would + Infinitive**           | If + Past Simple → would + Infinitive            | **If** I **were** you, I **would** talk to him. (Если бы я был на твоем месте, я бы поговорил с ним.)                        |
| **Третий тип (Third Conditional)**    | **If + Past Perfect**   | **would have + Past Participle** | If + Past Perfect → would have + Past Participle | **If** she **had studied** harder, she **would have passed** the exam. (Если бы она училась усерднее, она бы сдала экзамен.) |
| **Смешанный тип (Mixed Conditional)** | **If + Past Perfect**   | **would + Infinitive**           | If + Past Perfect → would + Infinitive           | **Had** I studied harder, I **would be** passing the exam now. (Если бы я учил(ся) усерднее, я бы сдавал экзамен сейчас.)    |


| **Форма глагола**                                      | **Пример** | **Использование**                                                                  |
| ------------------------------------------------------ | ---------- | ---------------------------------------------------------------------------------- |
| **Infinitive (инфинитив)**                             | to work    | Начальная форма глагола, используется для выражения действия без указания времени. |
| **Base form (основная форма)**                         | work       | Основная форма, используется с модальными глаголами (can, must, should и т.д.).    |
| **Past Simple (прошедшее время)**                      | worked     | Обычное прошедшее время для действий в прошлом.                                    |
| **Past Participle (причастие прошедшего времени)**     | worked     | Используется в совершенных временах и пассивном залоге.                            |
| **Present Participle (причастие настоящего времени)**  | working    | Используется для образования времени Continuous (действие в процессе).             |
| **3rd Person Singular (3-е лицо, единственное число)** | works      | Используется для 3-го лица единственного числа в Present Simple (he works).        |
